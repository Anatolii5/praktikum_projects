### Описание проекта:

В проекте мы  проанализировали данные об использовании услуг компании виртуальной телефонии, а именно:
 - информацию о новых клиентах
 - информацию о звонках данных клиентов

Клиентами компании являются организации, которым нужно распределять входящие вызовы на различных операторов(работников колл-центра), либо осуществлять исходящие вызовы силами операторов. Также операторы могут осуществлять внутренние вызовы, то есть вызовы друг между другом, которые осуществляются внутри сети виртуальной телефонии.

Основная цель проекта - это найти клиентов, которые переплачивают на текущих тарифах. Есть вероятность, что клиенты, которые экономически неэффективно пользуются сервисом уйдут к конкурентам. Таким образом можно понизить уровень оттока клиентов. Также исследование позволит лучше понять как пользователи потребляют услуги и возможно выдвинуть гипотезы нового ценообразования.

В ходе анализа данных провели следующие исследования: 
- выявили активных клиентов
- для каждого тарифа помесячно рассчитали:
	- количество клиентов 
	- продолжительность и количество звонков
	- количество входящих и исходящих звонков
	- количество внутренних и внешних звонков
	- количество пропущенных и отвеченных звонков
	- среднюю продолжительность звонка в день
	- количество звонков в день
	
Для поиска клиентов, которые переплачивают, провели следующие расчеты:
- проанализировали продолжительность внутренних исходящих звонков в месяц
- для каждого тарифа рассчитали количество минут, превысив которое выгоднее станет пользоваться другим тарифом
- проанализировали продолжительность внешних исходящих звонков в месяц и выявили клиентов, которые переплачивают

Результат проекта представляет собой общие данные, рассчитанные в ходе исследования, список клиентов, которые переплачивают, рекомендации по изменению тарифной линейки компании.

Используемые библиотеки для анализа данных:
- pandas
- numpy
- datetime
- seaborn
- matplotlib.pyplot
