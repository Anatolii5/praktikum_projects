### Описание проекта
В проекте мы  проанализировали данные анкет клиентов фитнес-центра.

Цели проекта:
- Спрогнозировать вероятность оттока для каждого клиента.
- Сформировать типичные портреты пользователей и охарактеризовать их основные свойства.
- Проанализировать основные признаки, наиболее сильно влияющие на отток.
- Cформулировать рекомендации по удержанию клиентов.

В процессе исследования данных:
- мы выявили признаки наиболее сильно влияющие на отток клиентов.
- построили матрицу корреляции и выявили мультиколлинеарные признаки.
- построили модель прогнозирования оттока клиентов, используя алгоритмы:
	- LogisticRegression
	- RandomForestClassifier
- провели кластеризацию пользователей и выявили характерные признаки для каждого кластера

Результат проекта представляет собой модель прогнозирования оттока, набор признаков, наиболее сильно влияющих на отток клиентов, рекомендации по удержанию клиентов.

Используемые библиотеки для анализа данных:
- pandas
- seaborn
- matplotlib.pyplot
- numpy
- sklearn
- scipy
