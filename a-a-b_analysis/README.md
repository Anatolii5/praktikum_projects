### Описание проекта

В проекте мы  проанализировали данные мобильного приложения, а именно:
- логи действий пользователей в мобильном приложении

Основная цель проекта - это узнать как ведут себя пользователи мобильного приложения. Проанализировать результаты A/A/B-теста: выяснить есть ли статистически значимые различия в конверсии для каждого события.

В ходе исследовательского анализа данных, мы рассчитали:
- общее количество событий в логах
- количество уникальных пользователей приложения
- среднее количество событий на пользователя
- выяснили данными за какой период мы располагаем
- провели анализ воронки событий:
	- рассчитали конверсию пользователей для простой продуктовой воронки
	- рассчитали конверсии пользователей для последовательной продуктовой воронки

В ходе анализа A/A/B-теста были пройдены следующие этапы:
- проверка на правильность механизмов и расчетов при проведении эксперимента:
	- проверили конверсию на стат. значимость для контрольных A/A групп.
- проверка конверсии на стат. значимость для групп:
	- А1 и В
	- А2 и В
	- А1+А2 и В
- корректировка уровня стат. значимости
- запуск проверки с новыми уровнем стат. значимости

Результат проекта представляет собой итог проверки конверсии на статистическую значимость для объединенной контрольной группы и экспериментальной.

Используемые библиотеки для анализа данных:
- pandas
- numpy
- scipy
- math
- seaborn
- plotly
- matplotlib.pyplot
