### Описание проекта
В проекте мы  проанализировали данные интернет-магазина, а именно:
- список гипотез
- результаты проведения A/B-теста
	- данные о заказах пользователей
	- данные о посещении сайта пользователями
	
Основная цель проекта - это принять решение о приостановке или продолжении A/B-теста. Выяснить есть ли статистически значимые различия в конверсии, в среднем чеке пользователей.

Проект состоит из двух частей:
- В первой части мы провели приоритизацию гипотез с помощью критериев **ICE** и **RICE** .
	
- Во второй части проанализировали A/B-тест: 
	- Для каждой группы  посчитали кумулятивные значения выручки, среднего чека, конверсии.
	- Проверили данные на аномалии.
	- Провели тесты на статистическую значимость по сырым и по очищенным данным.

Результат проекта представляет собой список наиболее важных для проверки гипотез и выводы по анализу проведенного A/B-теста.


Используемые библиотеки для анализа данных:
- pandas
- numpy
- scipy.stats 
- matplotlib.pyplot
