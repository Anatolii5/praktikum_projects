### Описание и цели проекта
В проекте мы  проанализировали данные интернет-магазина, а именно:
- логи о посещении сайта пользователями 
- список заказов пользователей
- ежедневные расходы на рекламу для разных источников трафика

Основная цель проекта - это определить эффективность маркетинговых затрат.

В процессе исследования данных мы рассчитали пользовательские метрики, такие как: 
- посещаемость в день/неделю/месяц
- количество сессий в день
- среднюю продолжительность одной пользовательской сессии
- среднее время до первой покупки

В ходе нашего проекта мы использовали когортный анализ по месяцам/ пользовательским устройствам/ источникам трафика пользователей. 

Используя когортный анализ мы рассчитали:
- средний чек пользователя
- среднее количество покупок на пользователя
- бизнес-метрики: Retention Rate / LTV / CAC / ROI
- определили сколько денег потратили: всего / на каждый источник / по времени

Результат проекта представляет собой рекомендации об изменении маркетинговой политики в зависимости от сезона года, в отношении различных источников трафика, устройств пользователей.

Используемые библиотеки для анализа данных:
- pandas
- numpy
- matplotlib.pyplot
- seaborn
